# Simple Library Pick Up Schedule API

## Prerequisites

- Go 1.23.2 or higher
- Docker
- Make

## Getting Started

### Build with Makefile

```bash
make build
```

### Run with Makefile

```bash
make run
```

### Run Tests

To run unit tests (using Mockery for mocks):

```bash
make test
```

### Run app with docker compose

```bash
docker-compose up --build
```
